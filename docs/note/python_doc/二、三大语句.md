## 2.1 顺序语句

### 2.1.1 链式赋值(雷！狗都不用)

```python
a = b = c= 1
print(a,b,c)

a = b = c = {'age':99}
b['age'] = 11

print(a,b,c)
```

    1 1 1
    {'age': 11} {'age': 11} {'age': 11}

### 2.1.2 try catch

    END

## 2.3 循环语句

### 跳出循环 break、continue

### while 循环

```python
num = 0

while num < 3:
    num +=1
    print(num)
else:
    print('end')
```

    1
    2
    3
    end

### for 循环

```python
nums = range(1,10)
for num in nums:
    print(num)
else:
    print('End')
```

    1
    2
    3
    4
    5
    6
    7
    8
    9
    End
