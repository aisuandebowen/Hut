    # 1.3 张量与Numpy

    # 1.3.1 张量转Numpy

    numpy() 二者会共享内存

    tensor1 = torch.ones(3,3)
arr = tensor1.numpy()

print(arr)
    1
    2
    3
    4
    [[1. 1. 1.]

    # 1.3.2 Numpy转张量

    • torch.Tensor(arr)

      会 复制 一份数据，创建新的张量，和原 NumPy 数组无关。

    • torch.from_numpy(arr)

      不会复制，新建的张量和原 NumPy 数组 共享内存，修改其中一个，另一个也跟着变。
